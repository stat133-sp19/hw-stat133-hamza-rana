Binomial Package

This package allows you to create and manipulate custom binomial variables and distributions. After binomial distributions are created, they can be visualized through graphs or can be printed. Functions of measures can also be summarized and outputted for each distribution. This package allows for the creations of two types of distributions: simple binomial distributions and cumulative distributions.

Creating a Binomial Distribution

# Creating the binomial variable
bin1 <- bin_variable(trials = 20, prob = 0.5)
bin1
#> "Binomial Variable"
#> 
#> Parameters
#> - number of trials: 20
#> - prob of success: 0.5

# Viewing summary statistics of the variable
summary(bin1)
#> "Summary Binomial"
#> 
#> Parameters
#> - number of trials: 20
#> - prob of success: 0.5
#> 
#> Measures
#> - mean: 10
#> - variance: 5
#> - mode: 10
#> - skewness: 0
#> - kurtosis: -0.1

# Statistics can be viewed singularly as well
bin_mean(trials = 20, prob = 0.5)
#> [1] 10

# Using bin_choose to calculate the number of combinations, which will be used to create the binomial distribution
bin_choose(n = 20, k = 5)
#> [1] 15504

# Calculating the probability value with certain parameters
bin_probability(success = 5, trials = 20, prob = 0.5)
#> [1] 0.01478577

# Creating a binomial distribution with the same parameters
